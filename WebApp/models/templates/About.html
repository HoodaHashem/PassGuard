About PassGuard

PassGuard is a password manager that helps you to store your passwords in a secure way. It is a free and open-source project that is developed by Mahmoud Hashem. The project is developed using Python and Flask. The project is hosted on GitHub and you can contribute to the project by forking the project and creating a pull request.

For more information about technology used and how it works please feel free to check out the README.md
